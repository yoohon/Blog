- 기능
: 기존 기능과 동일하지만 로직이 바뀌었다.
글쓰기와 수정은 기본적으로 동일하다.
차이가 있다면 사용자 입력 폼을 호출할 때, 글쓰기는 빈 상자만 출력하고 글 수정은 지정된 글번호에 대한 데이터를 조회하여
입력상자들의 초기값으로 지정하여 출력한다.
